# Automatic SEO

The system creates a collision-safe slug, SEO title, meta description, canonical URL, Open Graph title/description/image, robots directive, summary, tags, reading time, Article JSON-LD, and Breadcrumb JSON-LD.

The article renderer supplies the only H1. The body uses logical H2/H3/H4 levels. Drafts and failed work remain `noindex,nofollow`. Metadata supports discovery but cannot guarantee indexing or rankings.
