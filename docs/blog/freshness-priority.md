# Freshness Priority

Relevant, authoritative items published or materially updated inside 48 hours receive high editorial priority. Relevant items inside seven days are normally worth reviewing. Older items become low priority unless an evidenced rollout or material development is continuing.

The system stores the source publication date, source update date, discovery time, age in hours, priority reason, continuing-development flag, and expiry time. Freshness is an editorial label, not a ranking probability.
