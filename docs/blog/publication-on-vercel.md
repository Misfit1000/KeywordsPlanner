# Blog publication on Vercel

Publication is performed by Vercel server code after source, originality, metadata, image, prerender, and quality gates. Existing repository guards enforce review thresholds, emergency pause, publication pause, maintenance mode, and due time. The generation-job unique key prevents duplicate articles.

Public HTML, canonical, robots, Article and Breadcrumb JSON-LD, related links, responsive images, sitemap, news sitemap, and RSS remain served by Vercel. Drafts, previews, fixtures, and unpublished slugs remain noindex or 404.
