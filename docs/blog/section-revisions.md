# Section revisions

Editors can select the introduction, H2/H3 block, conclusion, tagline, summary, or meta description and request a bounded revision. Actions include shortening, improving clarity, adding a practical example, removing repetition, and rewriting from verified sources.

Without NVIDIA, the protected fixture provider demonstrates the same queue and revision model. The review panel shows current and proposed text, retained sources, changed claims, and quality results. Accepting a revision checks that the original section is unchanged and updates only that section. Rejecting preserves the article.
