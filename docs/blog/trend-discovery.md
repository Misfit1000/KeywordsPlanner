# Trend Discovery

The worker reads only administrator-approved public HTTPS RSS or Atom feeds. Feed fetches use DNS pinning, private-network blocking, redirect revalidation, response limits, and timeouts. Item URLs are re-fetched before generation.

Selection rejects stale, irrelevant, already-covered, weak-novelty, and overlapping opportunities. Two articles require distinct topic clusters and search intents. Google result pages are never scraped.
