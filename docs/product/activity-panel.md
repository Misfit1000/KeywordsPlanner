# Audit Activity Panel

The live event timeline is a collapsible, portaled panel rather than permanent report content. It is closed for first-time users, stores only an open/closed preference in local storage, and displays an unread count while closed.

Desktop uses a bounded right drawer; small screens use a bottom sheet. The panel has a dialog label, Escape handling, focus trapping and restoration, an independent event scroll area, and a backdrop close target on mobile. Event messages pass through the customer-safe diagnostic filter. Repeated crawl failures are summarized by stable failure category before finalisation.
