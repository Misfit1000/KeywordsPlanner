# Data Honesty

- Demonstration values must be labeled `Example report`, `Product preview`, or `Demonstration data` in the same visual context.
- Ranking, traffic, backlink, search-volume, and advertising values require a real import or provider.
- Crawlio Domain Strength is a transparent Crawlio metric built from measured audit categories. External link visibility and web rank are included only when available, missing evidence is excluded rather than scored as zero, and the confidence label reflects evidence coverage.
- External domain-signal lookups use attributed Majestic Million and Tranco data. They are not Moz Domain Authority, Ahrefs Domain Rating, a Google score, or a complete backlink index.
- Domain-signal lookups send only the normalized domain to the external providers. Account identity, audit findings, page metadata, and stored reports are not sent.
- Unavailable checks are excluded from pass counts and remain visible as coverage limitations.
- Audit scores describe collected checks; they do not predict rankings.
- Metadata previews are not live screenshots.
- Response-time observations are not browser-measured Core Web Vitals.
- Passive security findings are not evidence of exploitation or a penetration test.
- Complete raw page HTML is not retained.
